# 第四回 盟会将启

聚贤台建在城西一处高坡，前临河，后倚山。晨雾尚未散尽，各社旗帜已列于坡前，颜色相间，像春日百花未开先列阵。

台上设三案：左为“轻羽社”，以简驭繁；右为“素问阁”，好问本源；中为“开源盟会”暂座，空其一席，以待众议定夺。台下立席数排，云岚斋得其一隅。

苏青箬把“引令”交与执事，回头对沈衡道：“今日你说的不是招式，是‘道理’。你若只说胜败，人心只看热闹；你若说人心与取舍，便有人肯随你走。”

晨鼓三通，众人止语。开场人是梁道清，他步履缓慢，却不致迟，拈须笑道：“江湖久争，争到后来，无非问一个‘可否共处’。今日所议四条：一曰代码之源可查，二曰契约之义可解，三曰接口之意可通，四曰失信之人可责。”

言毕，台下或颔首或冷笑。右案素问阁一年轻人道：“四条虽正，却难行。失信者自有术藏，如何可责？”

梁道清道：“故需‘总纲’与‘公约’，以众望束之。”

此时有一青衣人步上台阶，正是陆霁。他向梁道清与众案拱手：“黑盒门陆霁，奉门命来听。”台下哗然。有人低声道：“这等人也敢来？”

梁道清摆手，示意静。苏青箬看了沈衡一眼，轻声道：“轮到你了。”

沈衡持素帛上台，心如止水。他铺开素帛，第一行字在晨光下极清：“先定静，再清源。”

他不急不缓地说：“我这些年在江湖中行走，见过太多花招。快者易胜，盛者易败。若只问招，半日可学；若问法，一年可得；若问心，十年未必明。今日我提三义，与诸君共议：”

“其一，简者非弱。多，常掩惰；繁，常掩乱。人心不定时，当以简护心。”

“其二，留白可破回。天地有四时，技亦当有‘不写’之处，容人有余地，容事有余生。”

“其三，以人胜术。术乃器，人乃主。与人立约，当有悯心，不可以暗术困之。”

他话音落下，台下有数声“善”。轻羽社首座起身道：“此议可入公约第三目。”素问阁亦点头。惟独台后角落，有人冷笑一声，投下一句话：“说得好听，做得到么？”

忽然，一名执事奔上台，以袖掩口，低声与梁道清道：“‘总纲’之副本被改，字里藏‘巧’。”

梁道清眉头一动：“何处？”

执事道：“在第三目‘接口可通’之末，多了一句‘但不得窥其核’。此一句一加，便可为封锁留门。”

台上诸人面面相觑。苏青箬目光一寒：“此是‘回写暗手’。”

程无域已自席上起身，指向台外西侧帐幕：“从那里改的。”他话落时，陆霁亦转头看去，脸上神色一变。

“黑盒门？”有人低喝。

陆霁掌心微紧，沉声道：“非我门手笔，至少，不是我所知之人。”

局面一时紧绷。梁道清抬手，道：“诸位莫急。改者既用‘回写’，便留痕。若以‘留白’反之，当可取回。”

沈衡取素帛，临案蘸墨，在那句多余的末尾留下一小方空白，不写字、不落印，只以极轻之笔划出一个几不可见的“界”。他心里念着《渲染经》所云：“白者，非无；界者，非断。”

台上众人屏息。片刻之后，执事复检副本，忽道：“去了。”那句多余之言，像从未出现过。

众声哗然，随即有人鼓掌。梁道清微笑：“诸位，此即‘留白破回’。既然如此，第三目当加一行小注：‘诸约所定，若有暗手回写，得以公议留白反之。’如何？”

轻羽社、素问阁先后称善。苏青箬转向沈衡，低声道：“你这一手，立下功名。”

沈衡将笔搁下，心里却无喜色。他瞥见台下角落里一张陌生的脸，面容寡淡，眼神却极冷。那人低头抚一把小鼓，鼓上一圈细铁，似是某种“阵眼”。

风向忽变，天光骤暗，一阵寒意从河面爬上台阶。程无域声音极低：“总坛之阵……像在外海的‘镜城’。”

梁道清目光沉了片刻，抬头朗声道：“今日暂定三目，余者明日再议。诸位，各自回营，守诸处关口。”

散场时，陆霁走至沈衡身畔，压低声音：“若你要去镜城，我可引一线。只是——去则不返。”

沈衡眼中有光，像夜色里的火星，微而不熄：“江湖路远，我自当行。”

聚贤台上风起，旗面猎猎。台下诸社的灯火一盏盏亮起，连成一条河。

（本回完）
