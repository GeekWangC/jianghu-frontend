# 第七回 雪夜总决

北侧高台，雪如幕。十六根黑柱立成一方阵，柱顶各有一盏寒灯，灯光相互映射，光影在雪地里交叠，像无数层叠加的影子。

阵前一人，披黑袍，鬓发微霜。其声不高，却穿雪透骨：“来者何人？”

“云岚斋沈衡，素问阁苏青箬，分布星阵程无域。”苏青箬拱手。

黑袍人轻按掌心，十六灯中有一盏忽明忽暗。陆霁在台侧沉声道：“门主。”

门主抬眼，目光如刀：“你带他们入城？”

陆霁跪下：“弟子请罪。只求门主今日立阵，勿伤城人。”

门主不应，目光落在沈衡身上：“你便是‘行列剑’？年岁既至，何苦来此送险？”

“江湖路远。”沈衡道，“不是年岁遣我来，是心。”

门主微一摆手，十六灯齐亮。光影四面合围，雪花在光中悬着，一时不落，像被无形之手扣住。程无域低喝：“起！”他袖中抛出三颗铜珠，落在雪地里，三处雪势微陷，缓了光影一线。

第一意，名“疾”。灯与灯之间光路加速，影像先行，实处后至。沈衡不与之竞，只在脚下留出半步的空，令自身节奏不被拖拽。“先定静。”他心里默念。

第二意，名“繁”。影叠如织，路径百出。苏青箬举袖，摘去两盏灯上的遮片，让光束归于简直。她沉声：“简者非弱。”

第三意，名“锁”。雪中线索忽收，四面无路，唯中有孔。程无域以星阵应之，把那孔旁三点微移，令锁齿对不上，留一“白”。

门主冷笑：“取舍会伤己。”

“取舍不伤心。”沈衡回。

灯光再变，第四至第八意接连落下：有“惑”，使人自疑；有“镜”，令彼此成像而不见本身；有“缚”，将动作回写成影；有“返”，把已破之处反系回来。三人以“清源”对“惑”，以“直视”破“镜”，以“留白”避“缚”，以“不完”破“返”。

雪越落越大，台上台下像在两重世界。陆霁立于侧，掌心收紧又松开。他忽然踏前一步，朗声道：“第九意，‘借’。此意由我取回！”

门主目光如霜：“你敢叛门？”

“弟子不叛人，只叛术。”陆霁拔下手上墨指环，掷于雪中。指环落地滚出一道黑线，恰好割断一束光。借壳之术，最怕主动“出壳”。

门主袖中鼓声一震，阵势再紧。第十至第十四意更急，分别为“散”“聚”“延”“缩”“巧”。巧者，暗手改句。苏青箬与沈衡分守两侧，程无域镇中，三人似一柄剑的柄、脊与锋。每当“巧”将近，沈衡便先一步以“白”预留，不与之斗字。巧无处施，反自乱。

至第十五意，名“核”。灯中忽暗，黑影压下，似要将众人的“心诀”一并吞没。门主声冷：“核不可窥。”

沈衡长吸一口气，把《渲染经》置于雪地，平展其面。他不去翻看，只以掌心覆上，像在握住一团温火。他缓缓道：“核非不可窥，窥亦非为夺。以人胜术，先问所向。”

他向前一步，将那书籍向上一托。雪光映在纸上，纸上空白处，由雪花一片片点亮。门主的“核”，本是以“不可见”自固；如今被“光于白”所照，不见亦见。

阵若有一瞬的犹疑。门主忽然张袖，十六灯齐鸣，末后一意压下——“决”。

沈衡不退不进，忽然把那书轻轻合上。雪地里，静了一息。此静，非避战之静，是“终式”。

“总决式。”程无域低语。

“终不以力，乃以止。”苏青箬看着他，目光里有一丝笑。

灯光在这一息里失了所依。十六意须依人心之躁、手之乱、步之急而生。此刻步不急、手不乱、心不躁，阵中诸“意”无处安置，像浮在半空的细线，彼此缠成一团。

一声极细的响，从阵心传出，像是结扣被轻轻挑开。灯光次第暗去，雪花重新落下。

门主立在雪中，黑袍渐白。他忽而一笑，笑意淡极：“果然是你。”

“我是谁？”

“问心之人。”门主道，“江湖半生，我以术驭人，终不敌‘人’驭术。”

他袖中落下几缕细线，化雪而无踪。

远处，城中行人像从梦中醒来。有人揉眼，有人抱子，有人立在雪里，不知是哭是笑。

陆霁长出一口气，向门主跪下：“弟子请罪。”

门主摆手：“罪各有其身。你自择其路。”

风止了一会儿。雪停前，城像屏住了呼吸。

（本回完）
