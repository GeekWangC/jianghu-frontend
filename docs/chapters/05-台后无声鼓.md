# 第五回 台后无声鼓

聚贤台散后，风声渐低，河面复静。唯有台后的空帐里，尚余一阵极轻的鼓声，像在胸腔里敲一记回响。

沈衡循声而至，帘缝之间，先见一只小鼓，鼓面镶着一圈细铁；再看，便见一个面容寡淡的男子，正以竹签轻点鼓缘。每一点，外头远处便有一丝人声错位、脚步错拍。

“无声鼓阵。”程无域在身侧道，“以时序扰人，以错位迷心。鼓声极轻，然一旦入耳，言语便多半不可信。”

男子抬目，眼神冷而无波：“既觉其术，何不退去？”

苏青箬入帐，抬手合上门帘，笑道：“阁下若真无声，何以在此击鼓？”

男子不答，只将竹签一折，刺入鼓边。鼓面轻颤，帐外立时有两名执事相向而行，险些相撞。男子淡淡道：“江湖爱热闹，我替它添些热闹。”

“你改了‘总纲’？”沈衡问。

男子手指在鼓边一敲，鼓声一顿，随后又起。他似笑非笑：“你们加了一条‘留白反回’，我便先行‘预写’。江湖上，写与回写，本是一事。”

“你从何来？”

“镜城。”男子收了竹签，起身拈鼓，“门主有命：若不能入‘核’之条，亦当使其余条自相牵制。”

苏青箬目光微凝：“你叫什么？”

“姓顾，单名一字，‘巧’。”

顾巧负鼓欲出帐，程无域侧身一挡。顾巧不退，鼓沿轻轻一刮，程无域肩头衣角随之颤了颤，仿佛筋脉被触。沈衡上前一步，掌心朝外，像在夜里挡住一阵乱风。

“顾巧。”他缓声道，“你鼓扰‘时’，我取‘静’。你移‘句’，我留‘白’。你若再敲，我便不破你术，只将此鼓挂于台上，让众人看见你所为。”

顾巧眼神一黯。半晌，忽抛鼓于席，冷声道：“我不与你争。镜城雪夜，你挡得住阵再言。”

他掠帘而出，人影如燕。

帐外风更清了。苏青箬拾起小鼓，递与执事：“挂台前，明示众人：今日之争，不在谁胜，而在不使暗术乱心。”

当夜，开源诸社草拟公约余目，素问阁添注“公开变更，留痕可查”。程无域一一画出关口布置，留出缓冲之“白”。

月落更深时，林拂衣来至台后，持一封未封的信，道：“黑盒门内，有人不愿‘核不可窥’，此信为他托我交你。”

苏青箬接过，目光复杂：“你怎会——”

“我在局内。”林拂衣笑意淡淡，“也愿在局外。”

信中只有一张小小的城图，在外海处圈出一岛，旁注“镜城·夜雪”。其下两字：陆霁。

沈衡收好城图，对苏青箬与程无域道：“两日后，海路。”

苏青箬点头：“我与程兄同去。”

风自河上来，带着一点潮寒。台前小鼓在夜色中泛着暗光，像一只被揭示了来历的影子。

（本回完）
