# 第一回 前路如剑

鹏城春雨，薄如绢纱。南面的高楼，一层层嵌着玻璃，映出行人影子，仿佛刀剑上褪不去的旧痕。午后时分，风里带着海腥，吹得街口的木槿微微摆首。

沈衡立在电梯口，手里只拎着一只旧背包。背包里铺着件灰蓝衬衫，压着一柄黑色小键盘。他把工牌放在前台的瓷碟里，朝姑娘笑笑，道：“烦劳。”声音不高，像在夜里掩门时的一缕气息。

人事姑娘名唤林拂衣，眼角眉梢有股冷清的劲。她把一叠纸翻过来又翻过去，递上一份协议，低声道：“沈工，流程如此。您……保重。”

沈衡“嗯”了一声，签了字。墙上大屏滚动着新一季目标，字字如金，与他再无干系。他想起昨夜修的那段性能，首屏像雪里探梅，终于在最恰当的时机绽放；又想起这几年里，诸派框架兴替如潮。有人成了盟主，有人遁隐江湖。唯有时间，缓缓走过，不动声色。

出了门，雨已停。天边露出些许蓝，像一柄藏锋的刀，未出鞘先逼人。

城南有座小馆，名为“渲染堂”。门檐低，茶汤好。沈衡一推门，便闻到橘皮与烘叶的清香。窗前坐着一位白须老人，衣衫旧而净，眉目宁静，像是等了许久。

“沈小友，别来无恙。”老人起身，拈须而笑。

“梁前辈。”沈衡拱手。他在江湖上混迹多年，识得几位长者。梁道清，人称“无穷客”，浪迹诸社，观法如看山川，淡淡评过，便叫人受用多年。

两人相对而坐。梁道清只问：“今日可得清闲？”

沈衡笑道：“清闲倒是清闲，只怕过于清闲。”

梁道清叹了一声，从袖中取出一本薄册。封面素白，只写三字：《渲染经》。他说：“此物早年得自一位奇人，非秘籍，亦非奇技，不过是些回到本原的‘招’。你这几年，刀法利落，却心火偏盛。读它，或可少走几步弯路。”

沈衡接过，拂去纸上的微尘。薄册启处，第一行便写：“观山不在高，先安其根。首屏者，如初式；动者，如呼吸。先定其静，再图其动。”他读得入神，只觉一股清凉之意，直从眼底入胸臆。

窗外雨脚又密。门口有青年束青衣入内，眉眼清朗，唇角却带轻薄之色。他手上把弄着一只墨色指环，指环上刻着细细密密的纹路，似是某种契约的印。

“这位便是‘行列剑’？”青年打量片刻，拱了拱手，笑而不笑，“久仰，久仰。在下黑盒门陆霁，奉命来请。”

梁道清不动声色，茶盖轻轻一合。

陆霁道：“江湖传闻，沈先生性能一诀，入木三分。如今盟内有一桩事，需请先生共襄。只是——”他顿了一顿，“敝门规矩，请者与被请者，先拆三招。若先生不愿，陆某也不强人所难。”

“何招？”沈衡把薄册合上，目光沉静。

“光影十六式中，拣最轻的一式。”陆霁笑道，“屏息、凝神，亮相即收。看谁的‘初式’更稳。”

茶馆狭小，客人稀稀。二人相对，竟似台上亮剑。沈衡缓缓伸手，仿佛按在虚空的一片水面。指尖微动，水纹层层散开。他心中默念《渲染经》所云：“先定静。”于是呼吸如线，气行丹田。忽听“嗡”的一声极轻，窗外天光如被拂拭，室内阴影顿浅，杯中茶色愈见清澈。

陆霁眼中闪过一丝意外。他指环一转，袖里几枚细银片无声落地，像是无形的“加载钩”。那是黑盒门拿手小术，先布钩，后牵线，凭借暗契牵引资源，不问来处。但见他食指微挑，桌面茶影忽而生出重影，影中又影，似有无数细线牵扯，将茶色拖得发滞。

梁道清低声道：“此招名‘缚影’，易伤经络。”

沈衡不语。他伸掌如护一炉微火，掌心微凹，指背带风，像有人在黑夜里替他擎着一盏灯。他念：“先定静，后清源。”气息一收，目光落在那几枚银片上，似看穿其藏处的路径。只见他掌缘轻拢，像将酒面浮沫一抹，重影顿消，茶色复清。

陆霁“咦”了一声，指环再转，袖中银片却已不见。他折扇一合，脸上笑意淡了几分，道：“先生好手段。只可惜，江湖路远，非一招可尽。”

“路远不怕，”沈衡道，“只怕心短。”

陆霁看了他一眼，忽然又笑：“先生若肯随我走上一遭，自有明白之处。黑盒门虽被人诟病，也有不得不为之由。”他话锋一转，声音低了半分，“再说，先生如今身在何处，浮沉难料。不若借我门一舟，避一时风浪。”

茶馆檐角滴水，滴在石阶上，声声入耳。

梁道清忽道：“陆小友，若只三招拆解，似乎尚欠一招。”

陆霁道：“前辈请。”

“第三招不在手上，在心上。”梁道清指着窗外，道，“这城风将转，夜里当有大雨。你我心里，可有伞？”

此言一出，陆霁沉默片刻。半晌，抱拳道：“受教。”他转身出门，青衣掠过檐下雨丝，背影明灭。

沈衡望着门外，忽觉得这天下宽广得很，楼宇如山，街道如河。人世间一切招数，到底都要落在“如何过这一段路”上。

梁道清把《渲染经》推了推：“拿去吧。你若不愿随他，便去云岚斋。那处清简，不问出处，只问心。”

沈衡把薄册揣入怀中，起身拎起旧背包。他对梁道清一揖，笑道：“多谢前辈。前路如剑，拭之则明。”

他推门而出，天色已暮。风起，雨未至，城中灯火如繁星初上。沈衡走入人潮，听见自己心里有一道极细的声音，像新弦初张，未尝一拨，已自清亮。

（本回完）
