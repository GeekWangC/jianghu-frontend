# 江湖前端：风起四十

一部以金庸笔法写就的当代武侠长篇：主角为奔四的前端工程师，在裁员风波与技术江湖的缝隙间，再入“江湖”。前端诸技化作武学门派，开源闭源如正邪两道，技术债与性能瓶颈犹如经脉暗伤。

## 项目结构
- `outline.md`：故事卷目与走向
- `characters.md`：主要人物卡、门派与关系
- `settings.md`：世界观与武学系统（技术→武学类比）
- `chapters/`：正文章节（按序号与标题命名）

## 章节命名规范
- 文件名：`NN-标题.md`，如：`01-前路如剑.md`
- 字里行间尽量保留“武林/门派/秘籍/内力/招式”等意象；
- 技术点入文应隐显相生：可化为门派禁招、经脉心法或兵器脾性。

## 写作建议
- 叙事参考金庸：以人物带动情节，以情义映照胜败；
- 招式描写与技术点一一成对，但不直白讲术；
- 每回末留钩子，或情或局，推动下一回。

## 开始阅读
- 从 `chapters/01-前路如剑.md` 开始。
